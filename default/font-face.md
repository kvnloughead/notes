# Example

```css
@font-face {
  font-family: 'Inter';
  src: url('./Inter-Black.ttf') format('truetype');
  font-weight: 900;
}
```

- The available types are: "woff", "woff2", "truetype", "opentype", "embedded-opentype", and "svg".
